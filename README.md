## Getting Started

1. **Install dependencies:**  
   ```bash
   npm install
   ```
2. **Run the development server:**  
   ```bash
   npm run dev
   ```
3. **For a production build:**  
   ```bash
   npm run build
   ```
4. **To preview the production build locally:**  
   ```bash
   npm run preview
   ```
